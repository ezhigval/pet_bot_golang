# Политика конфиденциальности ЗЕЛЕНИАЛ

Цель: обеспечить безопасность персональных данных пользователей и соблюдение законодательства РФ и международных норм.

## 1. Основные принципы

1. **Минимизация данных**
    - Сбор только необходимой информации для работы приложения.
    - Не храним лишние данные о действиях пользователя.

2. **Разделение персональных и обезличенных данных**
    - Персональные данные хранятся отдельно, шифруются локально на устройстве пользователя и на сервере.
    - Обезличенные данные используются для анализа и работы ИИ без раскрытия личности.

3. **Контроль доступа**
    - Доступ к персональным данным имеют только авторизованные процессы и сервисы.
    - Прямой доступ разработчиков к персональным данным отсутствует.

4. **Шифрование**
    - Все данные в базе шифруются (AES-256 или аналог).
    - Передача данных между клиентом и сервером — через HTTPS/TLS с дополнительными мерами защиты.

5. **Двухфакторная аутентификация (2FA)**
    - Пользователь при регистрации и входе может включить 2FA для повышения безопасности.
    - Поддержка Google Authenticator или SMS.

6. **Привязка устройств**
    - Каждое устройство привязывается к аккаунту.
    - Повторная регистрация с уже зарегистрированного устройства сокращает пробный период и генерирует алерт.

7. **Защита от взломов и пиратства**
    - Мониторинг частых попыток регистрации.
    - Ограничение бесплатных периодов.
    - Лояльные уведомления о подписках для честных пользователей.

8. **Сбор аналитики**
    - Собираются только обезличенные данные для анализа работы приложения и улучшения UX.
    - Личные данные не используются для рекламы или передачи третьим лицам.

9. **Соблюдение законодательства**
    - Соответствие требованиям ФЗ-152 РФ (о персональных данных).
    - Следование международным нормам GDPR для потенциальных пользователей за рубежом.

## 2. Практическая реализация

- Серверы с персональными данными изолированы.
- Обмен данными между микросервисами происходит с проверкой прав доступа.
- Логи приложения не содержат персональной информации.
- Регулярные аудиты безопасности и нагрузочного тестирования.

## 3. Права пользователей

- Доступ к своим данным.
- Исправление или удаление персональных данных.
- Отключение аналитики или отслеживания действий.
- Контактная поддержка для вопросов по конфиденциальности.
